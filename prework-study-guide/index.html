HTML

  • HTML is used to create the order of a webpage
  • An element is a type of content on a webpage
  • The head element contains information about the webpage
  • Elements that are nested inside other elements are called child elements

CSS

  • CSS defines the style of a webpage
  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Every HTML element is essentially a box
  • Do Not Repeat Yourself code is a coding principle that encourages developers to reuse of share code

Git

  • Git is a version control system
  • To access Git we use the command line
  • Git status: Checks what branch we are currently on
  • Git checkout -b branch-name: Creates a new branch and switches to it

JavaScript

  • A string is a sequence of text
  • Numbers don't have quotes around them
  • Everything in JavaScript is an object and can be stored in a variable
